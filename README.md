# Dark Path Prototype

Браузерный idle auto-battler: секторы, лут, события, боссы Рая и Ада.

## Запуск

Открой `index.html` в браузере или подними локальный сервер:

```bash
npx --yes serve .
```

## Управление

- **Пауза** — остановить автобой
- **1x / 2x** — скорость
- **♪** — музыка (MP3 в `assets/` или синтез)
- **Сброс** — очистить сохранение

Сохранение: `localStorage` ключ `dark-path-prototype-save-v2`.

## Ассеты

- `assets/hide-cs01-slowed.mp3` — фоновая музыка (опционально)
- `assets/locations/*.svg` — фоны локаций (уже в репозитории)

## Структура

- `index.html` — интерфейс
- `styles.css` — стили
- `js/` — модули игры (`main.js` — точка входа)
